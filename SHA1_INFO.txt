📋 ÉTAPES POUR CONFIGURER FIREBASE
═══════════════════════════════════════════════════════════════════

1️⃣ RÉCUPÉRER LE WEB CLIENT ID
   • Allez sur : https://console.firebase.google.com
   • Sélectionnez votre projet Craftus
   • Authentication > Sign-in method
   • Google > Modifier > Développer "Configuration du SDK Web"
   • Copiez le "Web client ID"
   • Format : 123456789-xxxxxx.apps.googleusercontent.com

2️⃣ MODIFIER capacitor.config.ts
   • Ouvrez : capacitor.config.ts
   • Remplacez 'VOTRE_WEB_CLIENT_ID.apps.googleusercontent.com'
   • Par votre vrai Web Client ID

3️⃣ AJOUTER L'APPLICATION ANDROID À FIREBASE
   • Firebase Console > Paramètres du projet
   • "Vos applications" > Ajoutez une application > Android
   • Package Android : com.craftus.app
   • Nom de l'app : Craftus

4️⃣ AJOUTER LES SHA-1 DANS FIREBASE
   • Dans les paramètres de l'app Android
   • Section "Empreintes de certificat"
   • Cliquez sur "Ajouter une empreinte"
   • Ajoutez le SHA-1 DEBUG : 87:42:27:AE:B5:69:CB:5A:3C:38:F8:1E:B6:40:90:74:0E:89:F5:97
   • Cliquez à nouveau sur "Ajouter une empreinte"
   • Ajoutez le SHA-1 RELEASE : 92:74:3E:78:87:38:9C:91:61:20:65:60:41:AD:24:C3:61:B7:A6:C4
   • ⚠️ Les DEUX empreintes sont nécessaires (debug pour tests, release pour production)

5️⃣ TÉLÉCHARGER google-services.json
   • Après avoir ajouté les SHA-1
   • Téléchargez le fichier google-services.json
   • Remplacez : android/app/google-services.json

6️⃣ REBUILD L'APPLICATION
   npm run build
   npx cap sync
   cd android && ./gradlew clean && ./gradlew assembleDebug
   cd ..
   adb install -r android/app/build/outputs/apk/debug/app-debug.apk

7️⃣ TESTER L'AUTHENTIFICATION
   • Lancez l'app : adb shell am start -n com.craftus.app/.MainActivity
   • Cliquez sur "Se connecter avec Google"
   • Une fenêtre NATIVE Android devrait apparaître (pas un navigateur)
   • Sélectionnez votre compte Google
   • ✅ Authentification réussie !
